Papers Presentation Format

  1. Papers in the following sessions are to be presented in a poster format:
    • MP8a - FPGAs and applications
    • MP8b - Statistical DSP applications issues
    • MN1 - Communication systems
    • TA8a - Radar and sonar II
    • TA8b - Applications of signal and information processing
    • TP8a - Wavelets and filterbanks
    • TP8b - Statistical array processing
    • WA8a - IIR and transform domain filters
    • WA8b - 2D and image processing applications

    All other papers are to be presented by oral presentation 20mn long (5mn questions allowed)

  2. Presentation Schedule:
    Paper identifiers consist of 2 letters, one number, and one optional letter. the first two letters identify the day of presentation, the number the session number, and the last letter identifies split sessons.

    • MA: Monday Morning
    • MP: Monday afternoon
    • MN: Monday evening (new format)
    • TA: Tuesday morning
    • TP: Tuesday afternoon
    • WA: Wednesday morning
